# 🚀 32-bit Single-Cycle RISC-V Processor

> A 32-bit Single-Cycle RISC-V Processor designed in **Verilog HDL**, verified using **ModelSim**, and implementing a subset of the **RV32I Instruction Set Architecture**.

---

## 📌 Project Overview

This project presents the design and implementation of a **32-bit Single-Cycle RISC-V Processor** based on the open-source **RISC-V RV32I ISA**.

The processor executes each instruction in a **single clock cycle** and was developed entirely in **Verilog HDL**. The complete datapath and control path were designed from scratch and verified through simulation using **ModelSim**.

The implementation includes arithmetic, logical, memory access, branch, and jump instructions, demonstrating the complete execution flow of a basic RISC-V processor.
